What is Shoutem?

App builder

Web dashboard used for creating and managing native apps by non-technical users

Platform for developers

Open-sourced, extendable, well-documented and based entirely on top of React Native

How does it work?

1. Create a new app

Use Shoutem Builder to create a blank app in no time or use one of our templates as a starting point

2. Add pre-developed features

Our 40+ open-sourced and editable modules called extensions allow you to reuse most commonly needed features, saving you time

3. Pull and edit app code

With Shoutem CLI you can pull the entire app code and continue on with development and changes locally
$ shoutem clone

4. Develop unique features

Shoutem is based on top of React Native allowing you to develop the rest of the app with pure native code in locally set up environment
$ shoutem push
